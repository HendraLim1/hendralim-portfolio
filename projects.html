Infrastructure Azure SAP B1 Networking
In Progress

Enterprise on-prem to Azure cloud migration.

Following a corporate acquisition, the entire IT infrastructure needed migrating from on-premises to the parent company's Azure environment — covering servers, SAP B1, email, and network infrastructure across 7 branches.

Problem

Post-acquisition, Australian Plastic Profiles operated on legacy on-prem infrastructure incompatible with Legrand's cloud-first ecosystem. Every system — from SAP B1 to email to physical networking — needed migrating without disrupting 300+ daily users.

Action
  • Planned and executed migration of 8+ on-prem servers to Legrand's Azure environment.
  • Migrated SAP B1 from on-prem to Azure cloud.
  • Completed Outlook/email migration for all users across 7 branches.
  • Designed VDI solution to resolve cross-branch latency issues.
  • Deployed FortiGate firewalls, network switches, and core switches.
  • Managed Active Directory and RDS environments throughout transition.
Result

Email migration fully completed. Network infrastructure deployed across branches. SAP B1 and remaining server migrations actively underway — progressing toward full integration with Legrand's ecosystem ahead of the June 2026 audit.

Scale: 300+ users · 7 branches · 8+ servers · Full network refresh — all while maintaining business continuity.
Software Modernisation .NET Cloud Readiness
Completed

Legacy .NET 2.0 → .NET 10 modernisation.

A business-critical application running on an end-of-life framework — vulnerable, unsupported, and blocking cloud migration. Fully modernised to .NET 10 while preserving all functionality.

Problem

The application ran on .NET Framework 2.0 — a platform that reached end-of-life with known security vulnerabilities, no vendor support, and no compatibility with modern cloud infrastructure or deployment pipelines.

Action
  • Audited the full codebase to map all dependencies and breaking API changes.
  • Rewrote and refactored application code to .NET 10 standards.
  • Updated all deprecated APIs, libraries, and third-party packages.
  • Tested thoroughly for regression, compatibility, and performance.
Result

Application fully modernised to a supported, actively maintained framework. Security exposure eliminated. Cloud deployment compatibility unlocked. Long-term maintenance overhead significantly reduced.

From EOL to current: .NET Framework 2.0 (2005) → .NET 10 (2024) — eliminating a 19-year-old security liability.
GRC ISO 27001 Compliance
Preparing

Internal audit preparation — post-acquisition context.

Applying ISO 27001 knowledge to prepare a newly-acquired business for its first internal audit under the parent company's compliance framework — coordinating across teams, documenting controls, and closing gaps.

Problem

Following Legrand's acquisition of Australian Plastic Profiles, an internal audit was scheduled for June 2026. The acquired entity had no established compliance documentation aligned to Legrand's security standards — requiring a full gap analysis and remediation effort.

Action
  • Applying ISO 27001 framework to structure audit preparation.
  • Reviewing existing security controls and identifying compliance gaps.
  • Documenting policies, procedures, and evidence packages.
  • Coordinating with stakeholders across departments to gather evidence.
  • Preparing remediation plans for identified gaps.
Result

Audit preparation on track for June 2026. Structured documentation in progress — controls mapped, gaps identified, and remediation plans underway. First audit of this kind for the acquired entity.

Context: ISO/IEC 27001 Provisional Auditor certification (PECB, Feb 2025) applied directly to a live enterprise audit scenario.
