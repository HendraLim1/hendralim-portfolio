Software Modernisation .NET
Completed

Legacy .NET 2.0 to .NET 10 migration.

Problem

Legacy application running on .NET Framework 2.0 — an end-of-life platform with known security vulnerabilities, no vendor support, and incompatibility with modern infrastructure and deployment pipelines.

Action
  • Audited the existing codebase to map dependencies and breaking changes.
  • Rewrote and refactored application code to .NET 10.
  • Updated deprecated APIs and libraries.
  • Tested for regression and compatibility.
Result

Successfully modernised the application to a supported, secure framework — enabling cloud deployment compatibility and reducing long-term maintenance risk.

Infrastructure Azure SAP B1
In progress

Enterprise on-prem to Azure cloud migration.

Problem

Following a corporate acquisition, the entire IT infrastructure needed to be migrated from on-premises servers to the parent company's Azure cloud environment — including SAP B1, email systems, and networking.

Action
  • Planning and executing server migration to Azure.
  • Migrating SAP B1 from on-prem to cloud.
  • Completed Outlook/email migration.
  • Deploying new switches, firewalls, and core switches.
  • Ensuring business continuity during transition.
Result

Email migration completed successfully. Network and SAP migrations actively underway — progressing toward full integration with the parent company's ecosystem.
